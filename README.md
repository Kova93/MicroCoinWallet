## How to build

1. Clone git repository
2. Init submodules
3. Run lazbuild MicroCoinWallet.lpi

```Shell
git clone https://github.com/MicroCoinHU/MicroCoinWallet.git
cd MicroCoinWallet
git submodule init
git submodule update
lazbuild ./MicroCoinWallet.lpi
```
